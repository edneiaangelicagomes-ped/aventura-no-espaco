# Viagem Espacial Game Documentation

## Project Description
Viagem Espacial is an educational math game designed for 3rd grade students. The game aims to make learning math fun and engaging through interactive challenges and adventures in space.

## Features
- Progressive Web App (PWA)
- Installable on devices
- Offline support for uninterrupted gameplay
- Three exciting levels with increasing difficulty
- A final challenge to test overall math skills

## Game Structure
- **Narratives**: Each level tells a unique story that captures the player’s interest.
- **Levels**: The game is organized into three levels, each introducing new mathematical concepts.
- **Clues**: Players receive helpful clues to solve math problems throughout the game.
- **Code**: The game is developed using engaging code that supports interactive learning.

## Controls

- **Web Browser**: Simply visit the game URL in any modern web browser to start playing.

## Game Mechanics and Level Descriptions
- Level 1: Introduction to basic addition and subtraction.
- Level 2: Understanding multiplication and division through fun challenges.
- Level 3: Advanced concepts including fractions and basic geometry.
- Final Challenge: A mix of all concepts learned in previous levels.
